README DEL PROYECTO

Título: Juego de la Búsqueda del Tesoro (futbol)

Descripción:
Este proyecto es un juego interactivo en el que el jugador debe mover un balón por un tablero de 10x10 hasta llegar 
a la portería. Cada turno se lanza un dado que determina las posibles celdas a las que puede moverse. El objetivo es 
alcanzar la portería en el menor número de lanzamientos posible.

Características principales:
- Validación del nombre del jugador antes de iniciar la partida
- Tablero dinámico de 10x10 con posiciones numeradas
- Tiradas de dado con imágenes representativas
- Movimiento del balón según el resultado del dado
- Mensajes de bienvenida, victoria y récord
- Almacenamiento del récord en LocalStorage para mantener la mejor marca

Tecnologías utilizadas:
- HTML5
- CSS
- JavaScript 
- LocalStorage

Instrucciones de uso:
1. Introducir un nombre válido y pulsar el botón Jugar
2. Pulsar Tirar dado para generar un número aleatorio
3. Seleccionar una celda resaltada para mover el balón
4. Repetir el proceso hasta llegar a la portería
5. Intentar superar el récord de lanzamientos

Buseif.
